# 🍽️ Recipe Bot

Телеграм-бот для хранения, просмотра и поиска кулинарных рецептов. Построен на [aiogram 3](https://docs.aiogram.dev/).

## Возможности

| Функция | Описание |
|---|---|
| 📖 Рецепты | Листание рецептов по категориям с навигацией ⬅️ ➡️ |
| 🔍 Поиск | Поиск по названию блюда или ингредиенту |
| ⭐ Избранное | Сохранение рецептов, листание избранных с навигацией |
| 🎲 Случайный | Случайный рецепт из всей базы |
| ➕ Добавить | Пошаговое добавление рецепта с фото, временем и сложностью |
| ✏️ Редактирование | Изменить любое поле рецепта прямо из карточки |
| 🗑️ Удалить | Удаление рецепта прямо из карточки (с подтверждением) |
| 📂 Категории | Просмотр, создание и удаление пользовательских категорий |
| /help | Встроенная справка |

## Быстрый старт

### 1. Клонировать и перейти в папку
```bash
git clone <repo-url>
cd recipe_bot
```

### 2. Создать виртуальное окружение и установить зависимости
```bash
python3 -m venv venv
source venv/bin/activate        # macOS / Linux
# venv\Scripts\activate         # Windows

pip install -r requirements.txt
```

### 3. Настроить токен
Скопируйте `.env.example` в `.env` и вставьте токен от [@BotFather](https://t.me/BotFather):
```bash
cp .env.example .env
```

Откройте `.env` и замените значение:
```
BOT_TOKEN=ваш_токен_от_BotFather
```

### 4. Запустить бота
```bash
python bot.py
```

Бот работает пока открыт терминал. Для фонового запуска:
```bash
nohup python bot.py &
```

## Добавление рецептов вручную

Отредактируйте `recipes.json`. Каждый рецепт — объект:

```json
{
  "title": "Название блюда",
  "ingredients": "Список ингредиентов",
  "steps": "Шаги приготовления",
  "image": "images/photo.jpg",
  "category": "Завтрак",
  "cook_time": "30 мин",
  "difficulty": "🟡 Средний"
}
```

Встроенные значения `category`: `Завтрак`, `Обед`, `Ужин`, `Десерт`. Пользовательские категории можно создать через кнопку `📂 Категории` прямо в боте.  
Поле `image` может быть путём к локальному файлу, URL или `null`.
